/*--- GPU Particle Architecture ---*/

Ring Particle Animation Requirements 
  - Initial Position Image      // Calculated Once
  - Final Position Image        // Calculated Once
  - 
  - Wait (Sum of R, G, B, and A)  // Image - Static
  - Current Position              // Dynamicly Interpolated with factor
  - Color (RGBA)                  // Image - Dynamicly Assigned based on factor
  - Completion Factor             // clamp01((currentTime - wait) / LENGTH)

Ring Particle Shaders
  - vertex/frag particle

  - TBD. Likely animated perlin noise used for pseudorandom drift.


/*--- Pseudocode ---*/

Shader: Position Initial
Shader: Position Final
Shader: Data
  > Wait (R)                // Constant
  > Alpha (G)               // Fades out early
  > Brightness (B)          // Burst toward end
  > Factor (A)              // clamp01((time - Wait) / LENGTH)


/*--- Math ---*/

Dynamic Velocity Smoothing Equation (Used for each ring particle's path.)

    velocity = (1 - s(t))a + b(s(t))

    a    = starting velocity
    b    = ending velocity
    s(t) = Smooth interpolation from 0 to 1 given input t
    t    = time
